Projects

These are a few robots I have built or worked on. I enjoy the hands-on experience gained from these projects. Hope my projects inspire all who are interested in robotics


  • Simultaneous Localization And Mapping:
    A robot that can map an arena with landmarks using cost-effective sensors
  • Infinity 3.0:
    A planetary rover that can go over rough terrain. Inspired from the Martian rover Curiosity
  • Craziflie:
    A Micro Quadcopter which was interfaced with MATLAB and ROS to run any path-planning code
  • Robot Fish:
    A bio-inspired robot fish inspired from carps that can swim underwater and pursue underwater tragets
  • Material Handling Robot:
    An autonomous mobile robot with a robotic arm and universal gripper to pick up and relocate objects
